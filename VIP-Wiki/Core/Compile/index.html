Компиляция

  1. Для компиляции потребуется SourceMod Stable или SourceMod Dev
  2. Компилируем по Инструкции
  3. Директивы препроцессора:
    • VIP_VERSION - Версия плагина (например: "3.0 DEV #17")
    • DEBUG_MODE - Режим отладки (1 - включить, 0 - выключить). Включает подробный вывод в лог информации о работе плагина.
    • USE_ADMINMENU - Включение админ-меню для управления VIP (1 - включить, 0 - выключить).
    • FIX_CSGO_MOTD - Включение фикса MOTD в CS:GO (1 - включить, 0 - выключить).
