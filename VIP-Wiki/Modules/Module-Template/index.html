Имя

Описание

Модуль ...

Требования

  • VIP-Core не ниже v2.1.1

Установка

  • В vip_modules.phrases.txt добавить
"ValveGloves"
{
    "en"    "Gloves (Valve)"
    "ru"    "Перчатки (Valve)"
}
"ValveGloves_MENU"
{
    "en"    "Select Gloves (Valve)"
    "ru"    "Выбор перчаток (Valve)"
}
  • В VIP-группу добавить
"ValveGloves"        "1"

Переменные

cfg/vip/VIP_Имя.cfg

// Группа для тестового VIP-статуса
// -
// Default: "test_vip"
sm_vip_test_group "vip1"

// Через сколько времени можно повторно брать тестовый VIP-статус (значение зависит от sm_vip_time_mode) (0 - Запретить брать повторно)
// -
// Default: "3600"
sm_vip_test_interval "1"

// На сколько времени выдавать тестовый VIP-статус (значение зависит от sm_vip_time_mode)
// -
// Default: "120"
// Minimum: "0.000000"
sm_vip_test_time "3600"
