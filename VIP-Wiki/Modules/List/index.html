Список модулей

Основные

  1. Admin - Позволяет выдавать VIP-статус по Админ-флагам и Админ-группам. http://hlmod.ru/threads/vip-admin.31882/
  2. Admin-Privileges - Позволяет выдавать VIP-игрокам Админ-флаги, Админ-группы и иммунитет. http://hlmod.ru/threads/vip-admin-privileges.26429/
  3. Vips - Добавляет команду !vips, которая отображает всех VIP-игроков онлайн.

Технические

  1. Porting - Позволяет перенести VIP-игроков с различных типов баз данных, версий плагина и с других VIP-плагинов. http://hlmod.ru/resources/vip-porting.355/
  2. Stats - Предназначен для сбора статистики о используемом ядре, игре и сервере. http://hlmod.ru/resources/vip-stats.290/

Вспомогательные

  1. Test - Позволяет игрокам взять пробный VIP-статус. http://hlmod.ru/threads/vip-test.27806/
  2. Language - Позволяет VIP-игроку выбрать язык меню и сообщений. http://hlmod.ru/resources/vip-language.239/
  3. Knife Arena Disable - Отключает указанные VIP-функции когда игрок находится на Knife Arena. http://hlmod.ru/resources/vip-knife-arena-disable.577/
  4. Last Request Disable - Отключает указанные VIP-функции когда игрок учавстует в "Последнем желании" (Last Request). http://hlmod.ru/resources/vip-knife-arena-disable.577/
  5. Features Manager - Позволяет управлять VIP-функциями на определенных картах, раундах и разминке

Визуальные

Добавляющие различные эффекты, изменения интерфейса и других визуальных составляющих.

  2. Имя - Описание.

Функциональные

Имеют функциональное назначение.
  1. Имя - Описание.
  2. Имя - Описание.
  3. Reserved Slot - Резервный слот для VIP-игроков. http://hlmod.ru/threads/csgo-rezerv-slotov-dlja-vip.37178/

Другие

  1. Everyday Gift Status - Ежедневная организованная выдача VIP-статуса, путём подбора оригинального подарка, выпавшего со случайного мёртвого игрока.

  2. SourceBans Expired Time - Синхронизирует время истечения VIP-статуса с плагинами администрирования. http://hlmod.ru/threads/vip-sourcebans-expired-time.34537/

  3. Имя - Описание.
