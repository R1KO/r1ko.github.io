Обновление

  1. Скачать архив с актуальной версией модуля:
    • GitHub
    • HLMod
  2. Распаковать архив.
  3. Заменить файл .smx на новый. (Если имена отличаются - старый удалить, а новый загрузить)
  4. Внимательно прочитайте список изменений новой версии и сопровождающую информацию:
    • Если там указано обновить какие-то файлы - сделайте это.
    • Если упомянуты изменения каких-то файлов - обновите и их.
  5. Выполнить одно из действий:
    • Перезагрузить сервер
    • В консоль сервера: sm plugins reload "vip/Имя_модуля.smx"
    • В консоль сервера: sm plugins refresh
    • Сменить карту
