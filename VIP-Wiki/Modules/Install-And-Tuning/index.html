Установка и настройка

  1. Скачать архив с актуальной версией модуля:

    • GitHub
    • HLMod

  2. Распаковать архив.

  3. Залить файлы на сервер соблюдая иерархию.

  4. Дописать фразы в файл перевода модулей addons/sourcemod/translations/vip_modules.phrases.txt (Написано в описании к каждому модулю. Если отсутствует - пропустите этот пункт)

  5. Настроить конфиг модуля. Находится addons/sourcemod/data/vip/modules/. (Написано в описании к каждому модулю. Если отсутствует - пропустите этот пункт) Само имя файла конфигурации модуля указывается в теме с модулем. За частую файлы конфигурации имеют структуру KeyValues (ключ - значение).

  6. Прописать нужным VIP-группам параметры, которые добавляет модуль. Находится addons/sourcemod/data/vip/cfg/groups.ini (Написано в описании к каждому модулю. Если отсутствует - пропустите этот пункт)

  7. После запуска модуля, будет создан конфиг. Находится cfg/vip/. (Написано в описании к каждому модулю. Если отсутствует - пропустите этот пункт) Само имя файла конфигурации модуля совпадает с именем плагина или указывается в теме с модулем. Эти файлы генерируются автоматически при первом запуске плагина.
